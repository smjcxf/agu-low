`scanner.py quick` 的实现很直接：**把日K线数据量从默认减少到60天，其他逻辑不变。**

```python
# scanner.py L2727-2731
elif cmd == "quick":
    # 盘中极速扫描(只要60天K线, 速度加倍)
    import __main__
    __main__.DAILY_BARS = 60
    scan_market()
```

- **触发方式**：命令行 `python scanner.py quick`
- **适用范围**：全市场扫描（创业板Top100 + 科创板Top100 + 主板Top100 + 港股Top50），不是只扫金股池

- **使用场景**：盘前09:15的快速更新任务（"小九备-09:15 盘前更新"）

如果你想要一个更轻量的版本（只扫金股池增量），可以用 `python scanner.py watch`（盘中精监金股池模式）。
